## working procedure / setup procedure
this file may not work in your pc if encrypt_decrypt.py file didn't run then follow below procedure
steps
1.open project directory
2.look for venv folder
3.open terminal or commandline in the project folder
4.run belows command for mac or linux
    source ./venv/bin/activate
5.if you cant run above command then look into google for how to activate virtualenv in your Operating System.
6 windows venv active code:
    venv\Scripts\activate
7 if you able to active venv then run the encrypt_decrypt.py file.
now you will be able to run the file.
